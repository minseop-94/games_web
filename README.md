# games_web
## 단편게임모음집

### 프로젝트 목적, 동기 
- (자바를 배운 김에)JAVA Web Application을 만들어 보고 싶었다. 
- 재미있고, 가벼운 게임들을 모아놓은 단편 게임집. 
- 번거로운 로그인은 없으면 좋겠다. 

### 프로젝트 진행 기간 
2021.12~2021.12 약 2주간 

### 스택, 툴 
- JAVA 
- Servlet
- Jsp 
- MySQL

- Eclipse
- Apache-Tomcat

### 보왼, 아쉬운 점 
- table 설계할 때, 다수의 인원을 한 그룹으로 묶어서 게임을 진행하고자 했으나, 시간이 부족해서 진행하지 못함
- 실제 배포를 하려고 계획했으나, 저작권 문제가 걸려 하지 못 함.
- 프로젝트 진행 중 인원의 유출이 발생해 계획대비 40%만 완성함. 

### 마무리 
프로젝트를 진행하다가 예상치 못한 인원의 누수, 에러가 많이 발생해서 짜증나고 열받고 했지만... 
지나고 보니 코딩, 프로젝트가 재밌다고 느꼈다.
